## 第1条（本アプリの特性と特別な同意事項）
本アプリは、ユーザーの集中力を高めるための「シールド機能（他アプリのブロック・オーバーレイ表示）」を有しています。ユーザーは本機能の仕様を十分に理解し、**自己の責任において本アプリを利用することに同意するものとします。**

シールド機能の展開中は、スマートフォンの画面上にオーバーレイが表示され、他アプリの操作が制限されます。緊急の電話着信、アラーム、災害速報（Jアラート等）などは原則としてOSの仕様に準じて表示されますが、シールド機能の性質上、またはユーザーの端末設定や環境によっては、これらが視認・認知しにくくなる、あるいはデバイスの操作が遅れる場合があります。

ユーザーは、緊急時やデバイスの操作が必要な場合には、本アプリ内の「ギブアップ」ボタンを押すか、タイマーを停止することで、いつでも即座にシールド機能を解除できることを確認し、これを理解した上で利用するものとします。

# 「Full Focus Timer」プライバシーポリシー

本アプリ「Full Focus Timer」（以下「本アプリ」）における、ユーザーの個人情報および利用者情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

## 1. 収集する情報および利用目的
本アプリでは、以下の情報を取得し、それぞれの目的で利用します。

**(1) ユーザーが入力・設定するデータ（ローカル保存）**
* **取得情報:** プリセット名、タイマーの各種設定時間、選択された環境音、オンボーディングの完了状態など。
* **保存方法:** これらのデータは、ユーザーの端末内（ローカルストレージ）にのみ保存されます。開発者のサーバー等へ外部送信・収集されることは一切ありません。
* **利用目的:** 本アプリの基本機能（タイマー設定の復元等）を提供するため。

**(2) 広告配信のためのデータ（外部送信）**
本アプリは、アプリの提供・維持（広告収益の獲得）を目的として「Google AdMob」による広告を導入しています。これに伴い、ユーザーの端末から直接、以下の情報が第三者へ自動的に送信されます。
* **取得情報:** デバイス識別子（Advertising ID / 広告ID）、IPアドレス、クラッシュログ、本アプリの利用状況（タップ履歴等）。

## 2. 電気通信事業法に基づく外部送信規律の表示
前項(2)に関連し、電気通信事業法の定めに従い、外部送信に関する事項を以下の通り公表します。

* **送信される情報:** デバイス識別子、IPアドレス、クラッシュログ、本アプリの利用状況
* **送信先:** Google LLC (Google AdMob)
* **送信先（Google LLC）における利用目的:** ユーザーの興味関心に合わせた適切な広告（パーソナライズ広告）の配信、および広告の成果測定・不正防止等のため。
* **当方（開発者）における利用目的:** アプリ内への広告表示による収益化のため。
* **Googleのプライバシーポリシー:** https://policies.google.com/privacy
* **Googleによるデータ使用の仕組み:** https://policies.google.com/technologies/partner-sites

## 3. パーソナライズ広告のオプトアウト（無効化）について
ユーザーは、ご自身のデバイス設定から、パーソナライズ広告のための広告IDの利用を無効化（オプトアウト）、またはリセットすることができます。

* **Androidの場合:** 「設定」>「Google」>「広告」>「広告IDを削除」または「広告のパーソナライズをオプトアウト」を選択してください。
※設定方法はOSのバージョンにより異なる場合があります。

## 4. 個人情報の安全管理措置
本アプリでは、ユーザーの個人を直接特定できる情報（氏名、住所、電話番号、メールアドレス等）の入力要求および収集は一切行っておりません。また、アプリ内で生成された利用データはすべて端末内に留まる設計となっており、高いセキュリティを確保しています。シールド機能に利用する権限（他のアプリの上に重ねて表示）を通じて、本アプリが他のアプリの画面上の情報やユーザーの入力内容を読み取ることは一切ありません。

## 5. プライバシーポリシーの変更
開発者は、法令の改正や本アプリの仕様変更に伴い、本ポリシーを随時改定することがあります。重要な変更がある場合には、本アプリ内の通知等によりユーザーにお知らせします。
